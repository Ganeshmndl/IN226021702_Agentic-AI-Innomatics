# IN226021702 FastAPI Food Delivery App

Simple FastAPI application that simulates a food delivery service. It exposes endpoints for browsing menu items, placing and managing orders, maintaining a cart, and running basic searches, filters, sorting and pagination.

## Quick Start

1. Create a virtual environment (recommended)  
   python -m venv .venv
   .venv\Scripts\activate

2. Install dependencies  
   pip install -r requirements.txt

3. Run the API locally  
   uvicorn main:app --reload

4. Open the docs  
   http://127.0.0.1:8000/docs

## Folder Contents

- main.py — FastAPI application
- requirements.txt — Python dependencies
- screenshots/ — example outputs from testing endpoints
